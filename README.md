# Simple Stack (C++)

A small C\+\+ project demonstrating:
- The Stack ADT in the form of a simple int stack
- Exception Handling


## Project structure

- `main.cpp` / `main.h`: application entry point and includes
- `stack.h` / `stack.cpp`: `Stack` class (pop, peek, push, isEmpty)
- `README.md`: this file

## Build and run (Linux)

From the project directory:

```bash
g++ -I ./ *.cpp
./a.out
```
